PACKAGING:

  ✓ Includes both kernel (lilux) and essential content (.ai/)
  ✓ Installation: pipx install rye (one command)
  ✓ Everything bundled, no separate installs

ONBOARDING:
  ✓ 17 guided demos across 4 phases
  ✓ Progressive complexity (simple → advanced)
  ✓ Hands-on learning (not theoretical)
  ✓ Fully runnable examples
  ✓ Real artifacts, not toys

EXECUTION:
  ✓ 4 core MCP tools (search, load, execute, help)
  ✓ Deterministic behavior
  ✓ Safety guarantees (recursion, cost, permissions)
  ✓ State files for persistence
  ✓ Knowledge mutation for learning
  ✓ Tight context packing for accuracy

STRUCTURE:
  ✓ Lilux = Kernel (stable, infrastructure)
  ✓ RYE = Content (essential, evolving)
  ✓ Together = AI-native execution
